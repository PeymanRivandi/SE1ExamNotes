# SE1 Exam Notes

A comprehensive study guide for Software Engineering 1 exams, presented as a responsive single-page HTML document.

## Contents

The notes cover the following topics:

- **Software Engineering Fundamentals** - Definition, process phases, development vs maintenance
- **Requirements** - Functional vs non-functional requirements, quality criteria, measurability
- **Verification & Validation** - Building the product right vs building the right product
- **Context Diagrams** - System boundaries, actors, interfaces (physical & logical), arrow directions
- **Glossary & UML Class Diagrams** - Domain classes, attributes vs relationships, multiplicity, inheritance
- **Use Case Diagrams** - Actors, use cases, «include» vs «extend» relationships
- **Software Testing** - Black-box testing (equivalence partitioning, boundary value analysis), white-box testing (coverage criteria)
- **Theories** - Waterfall vs Agile, architectural patterns, estimation, BPMN

## Features

- Mobile-responsive design with optimized table scrolling
- Print-friendly layout (2-column format for cheat sheets)
- Clean, modern UI with good readability
- Exam-focused content with memory aids and common pitfalls

## Usage

Simply open `index.html` in any web browser.

### Printing

Press `Ctrl+P` (or `Cmd+P` on Mac) to print. The document is optimized for a compact 2-column layout suitable for exam cheat sheets.
